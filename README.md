# 🤖 Telegram AI Discussion Bot

Умный Telegram-бот, который поддерживает диалоги на базе ИИ (через любой OpenAI-совместимый API, например OpenRouter / Gemini) и автоматически публикует вовлекающие опросы-дискуссии.

---

## 🌟 Основные функции

*   **💬 Интеллектуальный чат**: Общается с пользователями в ЛС и группах (через `@упоминание` или `reply`).
*   **🗳 Автоматические Дискуссии**: Дважды в день (по расписанию UTC через `APScheduler`) генерирует и отправляет горячие споры/опросы на случайные темы.
*   **🛡 Контроль затрат**: Лимит сообщений на пользователя в день (настраивается в `config.py`).
*   **🔄 Универсальный AI**: Легкое переключение мозгов бота (Gemini, Mistral, DeepSeek) простой правкой `.env` файла без изменения кода.
*   **💾 История диалогов**: Запоминает контекст беседы (последние 10-20 реплик).

## 🛠 Локальная установка и запуск

1. **Клонируйте репозиторий** (или скачайте):
   ```bash
   git clone https://github.com/Hayko19/TelegramBotAI.git
   cd TelegramBotAI
   ```

2. **Создайте и активируйте виртуальное окружение**:
   ```bash
   python -m venv venv
   # Для Windows:
   venv\Scripts\activate
   ```

3. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Создайте файл `.env`** в корне папки на основе вашего конфига:
   ```dotenv
   TELEGRAM_BOT_TOKEN=8776601169:AAF...ваш_токен
   CHAT_ID=-10024...ваш_чат
   
   # Настройки AI
   AI_API_KEY=sk-or-v1-...ваш_ключ
   AI_BASE_URL=https://openrouter.ai/api/v1/chat/completions
   AI_MODEL=mistralai/mistral-small-3.2-24b-instruct
   ```

5. **Запустите бота**:
   ```bash
   python main.py
   ```

## ☁️ Деплой (Запуск 24/7 в облаке)

Проект оптимизирован для работы на платформах **Railway.app** или **Render.com** через файлы `requirements.txt` и `Procfile`.

1. Привяжите скрытый репозиторий к Railway.
2. Добавьте переменные окружения (`Variables`) прямо на сайте.
3. Railway считает `Procfile` и запустит бота как фонового воркера (`worker`).

## 📁 Структура проекта

*   `main.py` — Точка входа, обработчик команд Aiogram, расписание APScheduler.
*   `ai.py` — Модуль отправки запросов к API и парсинга ответов.
*   `database.py` — Легкая БД `SQLite` (через `aiosqlite`) для учета лимитов.
*   `config.py` — Темы опросов, лимиты, системные промпты (настройки поведения).
*   `Procfile` — Инструкция для облачных серверов.
*   `.gitignore` — Список файлов, запрещенных к загрузке в интернет (ключи, База данных).
